# Tutorial: PMBOK Master

## Overview
Automate your project management workflow following Quản lý dự án (PM) standards.

## How to use
1. Input your project tasks in `project_plan.md`.
2. Update the `risk_log.json` as new threats are identified.
3. The agent will generate a Weekly Status Report based on these files.

## Key Logic
- Automatically calculates slack time.
- Flags overdue tasks in red.
