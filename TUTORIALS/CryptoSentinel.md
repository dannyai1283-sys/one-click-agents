# Tutorial: Crypto Sentinel

## Overview
Crypto Sentinel is designed to protect your portfolio by alerting you to sudden market shifts and sentiment-driven volatility.

## Setup
1. Define your assets in `agent.json`.
2. Connect your Telegram Bot API key.
3. Link your Twitter/X developer credentials for sentiment tracking.

## Running
Execute the agent using the blueprint runner:
`blueprint-run TEMPLATES/CryptoSentinel`

## Features
- **Price Alerts**: Immediate notifications for volatility.
- **Sentiment Analysis**: Understand the 'why' behind the move.
